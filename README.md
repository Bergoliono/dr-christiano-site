# Site institucional — Dr. Christiano Marinho Correia

Prévia profissional em Next.js + Tailwind CSS para cirurgião bucomaxilofacial em Uberaba/MG.

## Rodar localmente

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Build

```bash
npm run build
```

## Observações

- Textos com placeholders para CRO, WhatsApp, endereço e Google Maps.
- Linguagem institucional e ética, sem promessas de resultado.
- Projeto preparado para deploy na Vercel.
